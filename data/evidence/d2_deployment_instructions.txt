DEPLOYMENT TO VERCEL:
---------------------

Option 1: Vercel CLI
cd levqor-web
vercel --prod

Option 2: Vercel Dashboard
1. Go to vercel.com/new
2. Import Git repository containing levqor-web/
3. Framework: Next.js (auto-detected)
4. Root Directory: levqor-web
5. Build Command: npm run build
6. Output Directory: .next
7. Environment Variables:
   - NEXT_PUBLIC_ASSETS_BASE=https://api.levqor.ai
   - NEXT_PUBLIC_BACKEND_CHECKOUT=https://api.levqor.ai/billing/create-checkout-session
   - NEXT_PUBLIC_STATUS_URL=https://api.levqor.ai/ready
8. Deploy

Option 3: GitHub Integration
1. Push levqor-web/ to GitHub
2. Connect Vercel to GitHub repo
3. Auto-deploy on push
